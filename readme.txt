A plugin to include Oxygen editor content in Rank Math's content analsys.

== Description ==

This plugin enables [Rank Math](https://rankmath.com/) SEO plugin for WordPress to parse text inside the [Oxygen](https://oxygenbuilder.com) editor for Pages and Posts.

There are no settings for the plugin.

Just install and activate to make Rank Math include Oxygen editor content in its content analysis.

Note: For the updated/correct SEO score that includes the Oxygen editor content to be shown on Page/Post List screen, update the Page/Post.

== Installation ==

=== Automatic Installation ===

Search for `rank math oxygen` from within your WordPress plugins' Add New page and install.

=== Manual Installation ===

1. Click on the `Download` button to download the plugin.
2. Upload the entire `wpdevdesign-rankmath-integration-for-oxygen` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the `Plugins` menu in WordPress.
